# 秘密の部屋 機能詳細

## 概要
秘密の部屋は、primeNumber DATA SUMMIT 2025の管理システムにおける特別な高度管理機能を提供するセキュアな領域です。通常の管理者認証に加えて、追加のセキュリティレイヤーで保護されています。

## アクセス方法
1. 管理画面（/admin）にアクセスし、通常の管理者パスワード（PrimeData_Admin!2025）でログイン
2. 画面右下の紫色の「秘密の部屋」ボタンをクリック
3. 秘密の部屋専用のパスワード（secret2025）を入力

## セキュリティ保護
- 二重のセキュリティレイヤー:
  * 第1レイヤー: 通常の管理者認証（パスワード: PrimeData_Admin!2025）
  * 第2レイヤー: 秘密の部屋専用の追加認証（パスワード: secret2025）
- 認証状態はブラウザのlocalStorageに保存され、ブラウザを閉じるまで維持されます
- すべてのアクセスはログに記録されます

## 主要機能

### 1. システム設定
- サイト全体の設定を変更できる高度な機能
- 主な設定項目:
  * API接続設定（Supabase、OpenAI等）
  * サイト全体のメンテナンスモード切替
  * バックアップと復元
  * パフォーマンス最適化
  * エラーログ閲覧
  * キャッシュ管理

### 2. ユーザー管理
- 管理者アカウントの作成や権限設定機能
- 主な機能:
  * 管理者アカウント作成・編集・削除
  * 権限レベル設定（一般管理者/上級管理者/システム管理者）
  * アクセス履歴の確認
  * セッション管理
  * 2要素認証の設定
  * パスワードポリシーの設定

### 3. 高度な分析
- サイトのアクセス解析やユーザー行動の詳細な分析機能
- 主な機能:
  * リアルタイムアクセス状況
  * ユーザー行動パターン分析
  * コンバージョン率の詳細分析
  * ヒートマップ表示
  * A/Bテスト結果確認
  * カスタムレポート作成
  * データエクスポート
